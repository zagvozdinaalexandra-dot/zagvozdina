Основы кибербезопасности

Кибербезопасность состоит из трех основных столпов, которые защищают вашу информацию:

1. Конфиденциальность

Только авторизованные пользователи имеют доступ к вашим данным. Это означает, что ваша личная информация должна быть зашифрована и защищена от несанкционированного доступа.

  • ✓ Использование надежных паролей
  • ✓ Двухфакторная аутентификация
  • ✓ Шифрование данных
2. Целостность

Данные не изменены и являются подлинными. Это гарантирует, что информация не была изменена или испорчена без вашего ведома.

  • ✓ Проверка подлинности сообщений
  • ✓ Цифровые подписи
  • ✓ Контрольные суммы
3. Доступность

Системы и данные доступны, когда они вам нужны. Это значит, что ваши аккаунты не должны быть подвергнуты атакам типа DDoS или другим сбоям.

  • ✓ Резервные копии данных
  • ✓ Избыточные системы
  • ✓ Защита от DDoS-атак

Фишинг-атаки

Фишинг - это тип мошенничества, когда злоумышленники выдают себя за доверенное лицо или организацию для кражи ваших данных.

Типы фишинга:

Email фишинг

Поддельные письма, которые выглядят как из вашего банка, PayPal, социальных сетей и т.д. Они обычно содержат ссылки на поддельные сайты.

Признаки: Общие приветствия, срочность, подозрительные ссылки, прошу подтвердить личные данные
Spear Phishing

Целевые фишинг-атаки на конкретных людей или компании. Атакующие исследуют жертву и создают персонализированные письма.

Признаки: Знают ваше имя, детали работы, личные данные, более убедительные письма
Whaling

Фишинг-атаки на высокопоставленных должностных лиц (генеральные директоры, финансовые директоры и т.д.)

Признаки: Письма от якобы руководства компании с требованием передать конфиденциальную информацию

Как защитить себя от фишинга:

  • ✓ Проверяйте email адрес отправителя - часто они содержат опечатки
  • ✓ Наведите мышь на ссылки, чтобы увидеть реальный URL
  • ✓ Никогда не вводите пароли по ссылкам из писем
  • ✓ Включите двухфакторную аутентификацию
  • ✓ Проверяйте, защищен ли сайт (замок в адресной строке, https://)
  • ✓ Используйте специализированные фильтры от фишинга

Вредоносное ПО (Malware)

Вредоносное программное обеспечение - это программы, предназначенные для нанесения вреда вашему компьютеру или краже данных.

Типы вредоноса:

Вирусы

Самовоспроизводящиеся программы, которые заражают другие файлы и распространяются между компьютерами.

Черви (Worms)

Похожи на вирусы, но могут распространяться независимо без необходимости заражать другие программы.

Троян (Trojan)

Программа, выдающая себя за полезное ПО, но содержит вредоносный код.

Spyware

Программа, которая шпионит за вашей деятельностью и крадет личную информацию.

Признаки заражения малварью:

  • Компьютер работает медленнее обычного
  • Появляются всплывающие окна и реклама
  • Программы открываются и закрываются сами по себе
  • Большой объем трафика в сети
  • Изменения в браузере (новая домашняя страница, поисковая система)

Как защитить себя:

  • ✓ Установите и обновляйте антивирусное ПО
  • ✓ Скачивайте программы только с официальных сайтов
  • ✓ Не открывайте неизвестные вложения в письмах
  • ✓ Регулярно обновляйте операционную систему
  • ✓ Используйте файрвол

Программы-вымогатели (Ransomware)

Ransomware - это вредоносное ПО, которое шифрует ваши файлы и требует выкуп за их восстановление.

Как работает ransomware:

  1. Заражение через поддельное письмо, загруженный файл или уязвимость
  2. Программа шифрует все важные файлы на компьютере
  3. Появляется сообщение с требованием выкупа
  4. Чем дольше вы ждете, тем выше цена

Что делать, если ваш компьютер заражен:

  • ✓ Отключите компьютер от интернета сразу же
  • ✓ НЕ платите выкуп
  • ✓ Проверьте восстановление файлов из резервной копии
  • ✓ Обратитесь в поддержку соответствующих служб
  • ✓ Переформатируйте жесткий диск и переустановите ОС

Профилактика:

  • ✓ Регулярно создавайте резервные копии файлов
  • ✓ Отключайте макросы в документах
  • ✓ Используйте облачное хранилище для важных файлов
  • ✓ Обновляйте все программы и ОС

Атаки на пароли

Злоумышленники используют различные методы для взлома паролей и получения доступа к вашим аккаунтам.

Типы атак на пароли:

Brute Force (Перебор)

Программа пытается все возможные комбинации пароля. Слабые пароли можно взломать за часы или дни.

Dictionary Attack (Атака словаря)

Использование списка известных слов, имен и паролей для попытки входа.

Rainbow Tables

Предварительно вычисленные хеши паролей, используемые для быстрого нахождения пароля.

Credential Stuffing

Использование украденных учетных данных из взломанных сервисов для доступа к другим аккаунтам.

Признаки слабого пароля:

  • ✗ Менее 12 символов
  • ✗ Состоит только из букв или только из цифр
  • ✗ Используется на нескольких сайтах
  • ✗ Содержит личную информацию (день рождения, имя)
  • ✗ Является общеупотребительным словом

Признаки надежного пароля:

  • ✓ Минимум 12-16 символов
  • ✓ Содержит прописные и строчные буквы
  • ✓ Содержит цифры и специальные символы (!@#$%)
  • ✓ Уникален для каждого сайта
  • ✓ Не содержит личную информацию
  • ✓ Сложен для угадывания, но запоминаем для вас

Социальная инженерия

Техники манипуляции людьми для получения конфиденциальной информации или доступа к системам.

Распространенные методы:

Притворство (Impersonation)

Выдача себя за служащего компании, технического специалиста или авторитетного лица для получения доступа.

Претексты (Pretexting)

Создание выдуманного сценария для того, чтобы убедить человека раскрыть информацию.

Quid Pro Quo

Обещание выполнить какую-то услугу в обмен на информацию или доступ.

Tailgating

Физическое следование за сотрудником в защищенное помещение под предлогом доставки или встречи.

Как защитить себя:

  • ✓ Будьте осторожны с незнакомцами по телефону
  • ✓ Проверяйте личность человека перед тем, как делиться информацией
  • ✓ Не оставляйте пароли на стикерах или видном месте
  • ✓ Не делитесь личной информацией в соцсетях
  • ✓ Сообщайте о подозрительной активности
  • ✓ Пройдите обучение по информационной безопасности

Кража личных данных (Identity Theft)

Преступник использует вашу личную информацию для совершения мошенничества или получения кредитов.

Как воры получают доступ к вашим данным:

  • Взломанные базы данных компаний
  • Фишинг-письма
  • Поддельные сайты
  • Социальные сети
  • Потерянные или выброшенные документы
  • Открытые Wi-Fi сети

Признаки кражи личных данных:

  • ⚠ Счета, которые вы не открывали
  • ⚠ Неожиданные звонки от кредиторов
  • ⚠ Кредитные карты, которые вы не заказывали
  • ⚠ Странные платежи в выписках
  • ⚠ Отказ в кредите

Что делать при краже личных данных:

  1. Свяжитесь с вашим банком и кредитными учреждениями
  2. Подайте жалобу в полицию
  3. Заморозьте ваш кредитный счет
  4. Измените все пароли
  5. Отслеживайте кредитный отчет

Атаки "Man in the Middle" (MITM)

Злоумышленник перехватывает связь между двумя сторонами для кражи данных или внедрения вредоноса.

Сценарии атак MITM:

Wifi Sniffing

Атакующий создает поддельную Wi-Fi сеть и перехватывает трафик неосторожных пользователей.

DNS Spoofing

Перенаправление вас на поддельный сайт вместо настоящего через подделку DNS-запросов.

SSL Stripping

Удаление шифрования HTTPS и перехват незащищенного HTTP трафика.

Как защитить себя от MITM атак:

  • ✓ Избегайте публичных Wi-Fi сетей для конфиденциальных операций
  • ✓ Используйте VPN для защиты трафика
  • ✓ Проверяйте замок (https) в адресной строке
  • ✓ Не заходите в личные кабинеты через публичный интернет
  • ✓ Используйте двухфакторную аутентификацию
  • ✓ Проверяйте сертификаты SSL на сайтах
